### 명령어
```
전체 선택후 복사
- gg              : 어느 라인에서나 gg를 누르면 첫 줄로 이동한다
- (shift + v) + g : 첫줄에서 전체 선택하는 방법
- y               : 복사
- p               : 붙여넣기
- d               : 전체삭제
- set number      : Line number 보이기
- /               : 찾기
```
